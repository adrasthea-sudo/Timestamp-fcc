Timestamp Microservice

How Does This Work

This is a simple time converter api. It can convert any given time to two formats: the UNIX timestamp, which is the seconds since 1970 or just normal string date aka UTC format. You should put the date at the end of the url and it should be in one of the following formats:

/api/timestamp/2020-10-22
/api/timestamp/1603324800000

And the output will be a simple json object like this:

      
        {
          unix: 1451001600000, 
          utc: "Fri, 25 Dec 2015 00:00:00 GMT"
        }
